# 3D Printed Custom Enclosures & Mechanical Parts

## Project Overview

A comprehensive collection of 3D printed designs for electronics enclosures, sensor mounts, robot chassis components, and custom mechanical parts. Each design demonstrates practical problem-solving through additive manufacturing, combining functional requirements with aesthetic considerations.

## Role & Contributions

**Designer & Fabricator** - Complete design-to-production workflow:

- Analyzed project requirements and constraints
- Created 3D CAD models using Fusion 360
- Optimized designs for 3D printing
- Performed iterative prototyping
- Produced final parts on FDM 3D printers
- Documented design process and specifications

## Technologies & Tools

- **CAD Software**: Fusion 360, Tinkercad
- **3D Printing**: FDM (Fused Deposition Modeling)
- **Materials**: PLA, PETG, TPU (flexible)
- **Printers**: Ender 3, Prusa i3 MK3
- **Slicing Software**: Cura, PrusaSlicer
- **Post-Processing**: Sanding, painting, acetone smoothing

## Featured Designs

### 1. ESP32 Development Board Enclosure

**Design Objectives**: Protection, accessibility, professional appearance

**Features**:
- Snap-fit design for tool-free assembly
- Cutouts for all ports (USB, power, GPIO)
- Ventilation holes for heat dissipation
- Mounting holes for secure installation
- Stackable design for multi-module systems
- LED light pipes for status indicators

**Design Considerations**:
- Wall thickness: 2mm for structural integrity
- Tolerance: 0.2mm clearance for snap fits
- Print orientation for optimal strength
- Minimal support material requirements

**Applications**: IoT projects, development prototypes, educational kits

## Design Process

### 1. Requirements Analysis
- Identify project needs
- Measure existing components
- Define constraints (size, strength, aesthetics)
- Consider manufacturing limitations

### 2. Conceptual Design
- Sketch initial concepts
- Evaluate design alternatives
- Select optimal approach
- Plan assembly method

### 3. CAD Modeling
- Create 3D model in Fusion 360
- Apply parametric design for flexibility
- Perform virtual assembly checks
- Add tolerances and clearances

### 4. Design for Manufacturing
- Orient for optimal print quality
- Minimize support material
- Consider layer adhesion strength
- Add features for post-processing

### 5. Prototyping
- Print initial prototype
- Test fit and function
- Identify design issues
- Iterate improvements

### 6. Production
- Print final version
- Post-process (sanding, painting)
- Assemble with electronics
- Document design specifications

## Technical Specifications

### 3D Printing Parameters

**PLA (Standard Parts)**
- Layer height: 0.2mm
- Infill: 20-30%
- Print speed: 50mm/s
- Nozzle temp: 200°C
- Bed temp: 60°C

**PETG (Functional Parts)**
- Layer height: 0.2mm
- Infill: 30-50%
- Print speed: 40mm/s
- Nozzle temp: 235°C
- Bed temp: 80°C

**TPU (Flexible Parts)**
- Layer height: 0.2mm
- Infill: 50%
- Print speed: 20mm/s
- Nozzle temp: 220°C
- Direct drive recommended

### Design Guidelines Applied

- **Minimum Wall Thickness**: 1.5-2mm
- **Snap-Fit Tolerances**: 0.1-0.3mm
- **Threaded Inserts**: M3, M4 brass inserts
- **Maximum Overhang**: 45° without supports
- **Bridging Distance**: Less than 20mm
- **Hole Tolerance**: -0.1mm for press fits
