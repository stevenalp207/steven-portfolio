# PLC-Based Industrial Control System

## Project Overview

Designed and implemented a comprehensive PLC-based control system for industrial process automation. The system manages multiple motors, monitors process variables, and integrates with SCADA for visualization and remote control capabilities.

## Role & Contributions

**PLC Programmer & System Integrator** - Full responsibility for control system development:

- Analyzed process requirements and created functional specifications
- Designed control logic and state machine diagrams
- Programmed PLC using Ladder Logic and Structured Text
- Configured SCADA system for monitoring and control
- Implemented alarm management and data logging
- Conducted commissioning and testing

## Technologies Used

- **PLC Platform**: Siemens S7-1200 / Allen-Bradley CompactLogix
- **Programming**: TIA Portal, RSLogix 5000
- **Languages**: Ladder Logic, Structured Text, Function Block Diagram
- **SCADA**: WinCC, FactoryTalk View
- **HMI**: Industrial touch panel displays
- **Communication**: Profinet, Ethernet/IP
- **Simulation**: Factory I/O, PLCSim

## Technical Implementation

### PLC Programming Examples

#### Motor Control Logic
```
// Ladder Logic representation
START_BUTTON AND NOT_STOP AND READY --> MOTOR_RUN
MOTOR_RUN AND (OVERLOAD OR E_STOP) --> FAULT_STATE
```

#### PID Temperature Control
```
// Structured Text
TEMP_PID(
    PV := TemperatureSensor,
    SP := TemperatureSetpoint,
    KP := 2.0,
    TI := T#30s,
    TD := T#5s,
    OUTPUT => HeaterOutput
);
```

- **Problem**: Meeting strict cycle time requirements
- **Solution**: Optimized scan times and used interrupt-driven programming

### Challenge 3: Complex Interlocking
- **Problem**: Managing multiple safety and operational interlocks
- **Solution**: Created structured logic with clear documentation

### Challenge 4: Alarm Management
- **Problem**: Alarm flooding during fault conditions
- **Solution**: Implemented alarm prioritization and suppression logic
