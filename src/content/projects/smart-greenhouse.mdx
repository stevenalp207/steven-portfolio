# Smart Greenhouse Automation System

## Project Overview

An intelligent greenhouse automation system that maintains optimal growing conditions for plants through automated climate control, irrigation management, and real-time monitoring. The system uses IoT technology to enable remote monitoring and control via web and mobile interfaces.

## Role & Contributions

**System Designer & Developer** - Complete system architecture and implementation:

- Designed greenhouse sensor network and control systems
- Programmed microcontroller logic for automated control
- Developed cloud-based monitoring dashboard
- Implemented automated irrigation and climate control
- Created alerting system for critical conditions
- Conducted field testing and optimization

## Technologies Used

- **Microcontrollers**: ESP32 for WiFi connectivity and control
- **Sensors**: DHT22 (temperature/humidity), soil moisture, light intensity, CO2
- **Actuators**: Relay modules, solenoid valves, fans, heaters, LED grow lights
- **Communication**: MQTT protocol, WiFi
- **Cloud Platform**: Adafruit IO for data visualization
- **Programming**: Arduino C++, Python for data analysis
- **Power**: Solar panels with battery backup system

## Control Logic Implementation

### Climate Control Algorithm
```cpp
void controlClimate() {
    float temp = dht.readTemperature();
    float humidity = dht.readHumidity();
    
    // Temperature control
    if (temp > targetTemp + 2) {
        activateFans(HIGH);
        deactivateHeater();
    } else if (temp < targetTemp - 2) {
        activateHeater();
        activateFans(LOW);
    }
    
    // Humidity control
    if (humidity < targetHumidity - 10) {
        activateMisting();
    } else if (humidity > targetHumidity + 10) {
        increaseVentilation();
    }
}
```

### Smart Irrigation Logic
```cpp
void checkIrrigation() {
    int soilMoisture = readSoilMoisture();
    
    if (soilMoisture < moistureThreshold) {
        if (canIrrigate()) {  // Check time and weather
            activateIrrigation(calculatedDuration);
            logWaterUsage();
        }
    }
}
```
